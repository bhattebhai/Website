What is the Website about?

The website is an ecommerce website for store that sells varieties of printer. The website is names as DontPrint and has five pages; Home page, Product page, Blog Page, Research Page and About us page. Customers can buy their required type of printers through this website. They can signup and login to the website through the email and can leave the feedback for the improvement in coming days.

Our Website Design Reference

As an ecommerce website there are many websites in the internet. Some online stores or ecommerce website we visited before developing one for us are Amazon, Zomato, Flipkart, Myntra, Zara, H&M, Alibaba, Daraz and Shopify. We studied and analyzed the design that these websites used to make their website unique from each other’s. Each of them had unique design, different color combination and many differences. Also, they have some similar concepts while it comes to the design of header, footer or search box. Not exact similar but the design concept was too similar. The header contains the logo and search box as primary element and other elements like cart, location, day/night mode, language as their design. Similarly, almost all the website has similar footer elements like blog, about us, contact info, and other. In our overall design we used different ideas from different website.

Our Coding Reference

To develop the website, we took code reference through different websites. W3school, GeeksforGeeks, MDN Web Docs are some website that we used to learn about HTML, CSS and JavaScript. These websites have different practice questions and pre-built components used in web developing.
